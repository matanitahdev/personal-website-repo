Emergant

a reinforcement learning project that I built to learn the fundamentals of deep learning using PyTorch and CUDA. It is an ant colony simulation which uses highly computationally efficient reinforcement learning to allow for ant-like behaviors to naturally emerge. Now an open source contribution to the state-of-the-art library PufferLib!
